# 植物大战僵尸工业级重构开发计划

## 项目目标
将基础版植物大战僵尸重构为具备工业级品质的完整游戏，包含50+关卡、49种植物、26种僵尸、多场景支持及完善的系统架构。

4. **分支管理**: 所有开发在 master 主分支进行
